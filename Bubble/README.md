# Bubble

> A lightweight macOS menu bar app for managing and quickly accessing your prompt library.

## 快速上手

**功能一句话：** 在菜单栏点一下（或按 `Cmd+Shift+Space`）唤出提示词面板，搜索、编辑、复制你的提示词库。

### 特性

✨ **菜单栏应用** — 无 Dock 图标，不占桌面  
🌐 **全局快捷键** — 在任意应用中按 `Cmd+Shift+Space` 唤起（可自定义）  
🎨 **苹果风磨砂玻璃** — 原生 macOS 视觉风格  
🔍 **搜索+分类筛选** — 快速找到你的提示词  
✂️ **一键复制** — 复制到剪贴板后可立即粘贴  
📝 **自定义标签** — 用户自定义提示词分类  
⚙️ **灵活设置** — 开机启动、快捷键自定义、菜单栏入口开关

## 安装 & 运行

### 系统要求
- **macOS 14.0** (Sonoma) 或更新版本
- **Xcode 15.0+**（如需本地编译）

### 从源码编译

```bash
# 1. 进入项目目录
cd /Users/pengchonglin/Desktop/vibe\ coding/提示词小弹窗/Bubble

# 2. 用 Xcode 打开项目
open Bubble.xcodeproj

# 3. 选择 Bubble scheme，按 Cmd+B 编译，Cmd+R 运行
```

首次运行时，系统会提示需要"辅助功能"权限（用于全局快捷键）。  
点击"系统偏好设置" → "安全与隐私" → "辅助功能"，勾选 Bubble。

### 功能权限说明

| 权限 | 原因 | 首次申请时机 |
|------|------|------------|
| 辅助功能 | 全局快捷键 (Cmd+Shift+Space) 需要监听键盘 | 首次启动 |
| 开机启动 | 勾选设置中的"开机自启动" | 用户在设置面板点击时 |

## 使用指南

### 主要工作流

1. **打开面板**
   - 点击菜单栏的气泡图标
   - 或在任意应用中按 `Cmd+Shift+Space`

2. **搜索提示词**
   - 在搜索框输入关键词（搜索标题+内容）
   - 或点击顶部标签筛选（写作/编程/翻译/办公/营销/学习/自定义）

3. **复制提示词**
   - 点击卡片右侧的"复制"按钮
   - 按钮变为 ✓ 表示复制成功
   - 立即可在其他应用粘贴

4. **新建提示词**
   - 点击底部 `+ 创建` 按钮
   - 填入标题、内容（max 2000 字）、分类标签
   - 点击"保存"

5. **编辑提示词**
   - 在列表中点击卡片右侧的铅笔图标
   - 修改后点击"保存"

6. **删除提示词**
   - 点击编辑页面底部的"删除"按钮
   - 确认删除

### 设置面板

点击底部 `⚙️ 设置` 进入设置面板：

- **全局快捷键**：点击输入框，按下新的组合键（必须包含修饰键 like Cmd/Ctrl/Option）
- **开机自启动**：Toggle 开启后，Mac 重启时 Bubble 自动运行
- **显示菜单栏入口**：Toggle 关闭后，菜单栏图标隐藏（但快捷键仍可唤起面板）

## 数据存储

所有提示词数据存储在本地，使用 SwiftData（macOS 原生数据库）。  
**备份：** 数据库文件位于 `~/Library/Application Support/Bubble/`，可手动备份。

## 快捷键

| 快捷键 | 动作 |
|--------|------|
| `Cmd + Shift + Space` | 打开/关闭面板（全局，可自定义） |
| `Cmd + S` | 保存（在编辑表单时） |
| `Esc` | 关闭面板或返回列表 |

## 故障排查

### 快捷键无响应
- **原因**：未授予"辅助功能"权限
- **解决**：系统偏好设置 → 安全与隐私 → 辅助功能 → 勾选 Bubble

### 面板弹出位置不对（多显示器）
- **原因**：系统无法正确识别当前显示器
- **解决**：重启 Bubble 或重启 Mac

### 修改快捷键后没生效
- **原因**：新快捷键与其他应用冲突
- **解决**：在设置中查看警告信息，选择其他组合键

### 开机启动没工作
- **原因**：macOS 启动项权限问题（不常见）
- **解决**：关闭再开启设置中的"开机自启动" toggle
